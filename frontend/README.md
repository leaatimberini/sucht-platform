# SUCHT - Frontend

Este es el frontend de la aplicación de gestión de eventos SUCHT, construido con Next.js, TypeScript y Tailwind CSS.

---

## ▶️ Instalación y Primeros Pasos

Para correr este proyecto en un entorno de desarrollo local, sigue estos pasos.

### **1. Prerrequisitos**

Asegúrate de tener instalado [Node.js](https://nodejs.org/) (versión 20.x o superior).

### **2. Clonar el Repositorio**

```bash
git clone [https://github.com/leaatimberini/sucht-frontend.git](https://github.com/leaatimberini/sucht-frontend.git)
cd sucht-frontend
